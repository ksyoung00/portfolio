Publishing Guide

작업 전 퍼블리싱 가이드 작성

  • * 서체 : 구글 웹폰트를 cdn방식으로 연결/시스템 폰트 폴백으로 연결
  • * 선택자 작성 : 클래스명은 종속선택자로 작성하지 않음
  • * 색상은 기본 6자리 16진수, 축약가능한 경우 3자리 16진수
  • * Media Query : 모바일 우선으로 작업하고, 선언방식은 HTML내부에 Link연결
